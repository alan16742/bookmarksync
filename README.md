# WebDAV书签同步

一个简单的 Chrome 扩展，用于通过 WebDAV 在不同设备间同步浏览器书签。

## 功能特性

- 📚 支持书签的上传与下载
- 🔄 通过 WebDAV 协议同步
- 🔒 安全存储 WebDAV 认证信息
- 🧹 支持清空本地书签
- ✅ 支持连接测试

## 安装方法

1. 下载本项目代码
2. 打开 Chrome 浏览器，进入扩展程序页面 (`chrome://extensions/`)
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹

## 使用说明

1. 点击浏览器工具栏中的扩展图标
2. 在设置区域填写 WebDAV 服务器信息：
   - 服务器地址（例如：https://dav.example.com）
   - 用户名
   - 密码
3. 点击"测试连接"确保配置正确
4. 点击"保存设置"保存 WebDAV 配置
5. 使用"上传书签"或"下载书签"进行同步

## 注意事项

- 首次下载书签会清空本地已有书签
- 请确保 WebDAV 服务器支持 CORS
- 建议在同步前备份重要书签
